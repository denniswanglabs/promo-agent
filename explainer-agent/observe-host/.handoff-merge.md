## How to demo

1. Open Safari to `http://127.0.0.1:8081/` (per Dennis's localhost rule).
2. Type a start URL and goal.
3. Click "RUN AGENT" — submitting calls `:8082/api/run` and starts polling.
4. The status pill switches `idle → live (Discovering...) → live (Recording...)
   → live (Rendering...) → done`.
5. When done, the live frame view swaps to a looping `<video>` of the MP4.
6. The `frame.jpg` + `log.txt` continue updating from `poll.sh` independently
   throughout (the page polls both static files AND the job-specific log
   lines via `/api/status` — the job log takes over once `activeJobId` is
   set).
